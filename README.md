# TicTacToe 🎮

## Description
A simple CLI-based Tic-Tac-Toe game written in **C#**.

## Requirements
  + [.NET SDK](https://dotnet.microsoft.com/download)
  + [Git](https://git-scm.com/)

## Installation
Clone the repository:
```bash
git clone https://github.com/BaibhavPhaijoo/Tic-Tac-Toe.git
cd Tic-Tac-Toe
```
Run the game:
```bash
dotnet run
```
